### 4. 시험중 알아야할 명렁어

* Windows Copy
  * ctrl + insert
* Windows Paste
  * shift + insert
* kubectl
  * Resoruce API Version 확인 : kubectl api-resources
  * Resource Spec/Status 확인 : kubectl explain --recursive <resource>
